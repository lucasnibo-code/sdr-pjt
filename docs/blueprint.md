# **App Name**: SDR VoiceInsights

## Core Features:

- Call Data Ingestion: Securely upload or link to call recordings for processing and analysis within the platform.
- AI Call Transcription & Summarization: Utilize an AI tool to automatically transcribe call audio into text and generate concise summaries, highlighting key discussion points and sentiment.
- Call Evaluation Interface: Provide a clean user interface to review call transcripts, playback audio, and apply custom evaluation criteria or scores.
- Basic SDR Dashboard: Display a dashboard overview showing evaluated calls and basic performance metrics related to SDR activity.
- User Authentication: Allow users to register and log in to manage their call evaluations securely.

## Style Guidelines:

- Light color scheme, fostering a sense of clarity and focus, appropriate for analytical tools.
- Primary color: A professional and trustworthy blue (#336699) that conveys reliability for a corporate environment.
- Background color: A very light, subtle blue (#E6EDF2) provides a clean canvas without being stark white.
- Accent color: A soft, inviting aqua (#80CCCC) offers visual distinction for interactive elements and highlights, providing contrast to the primary blue while remaining corporate-appropriate.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, technically-aware, and impactful feel in titles.
- Body font: 'Inter' (sans-serif) for highly legible and neutral body text, suitable for detailed data and long-form content.
- Minimalistic and sharp line icons that reinforce the clean and professional aesthetic.
- A clean, grid-based layout with ample whitespace to enhance readability and organization of data, reflecting a corporate style.
- Subtle, non-distracting transition animations that provide feedback without overwhelming the user.
